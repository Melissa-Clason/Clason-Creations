Rogue Pickings: A Food Truck Website

I built this site using one of Skillcrush’s design comps as a way of practicing my HTML, CSS, and Photoshop skills. Using Photoshop to measure spacing, I was able to create a working version of the site with the same fonts, images, and styles from just a picture. This demonstrates that I have the skills to take your design and make it real.

Jubilee Austen: A Client Portfolio

This project was my first attempt at creating a personal website for a fictional client, a web developer named Jubilee Austen. It includes separate sections about her, about her skills, and how to contact her. With this project I learned the basics of intertwining HTML and CSS to create a layout with multiple defined sections.
